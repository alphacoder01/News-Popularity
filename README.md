# News-Popularity
A xgboost based classifier for prediction viral news based on UCI-News popularity dataset.

## Classifier Used:
`xgboost with parameter tuning`.

## Dataset:
`UCI-News Popularity Dataset`

## Accuracy:
`67.36% on the test-set.`

## Dependencies:
1. Xgboost
2. numpy
3. pandas
4. scikit-learn.
